# ML-PROJECT
 Progetto di ML/SII per UniRoma3

Cross Recommandation: Raccomandare all’utente una lista di film collegati a una certa locazione geografica, ordinata in base alle sue preferenze. 

Dataset non inclusi: ottenibili da https://grouplens.org/datasets/movielens/, file ml-latest.zip

In jsons è possibile visualizzare il risultato della query SPARQL, salvata per comodità in fase di test.
